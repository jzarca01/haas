# Usage
<!-- usage -->
```sh-session
$ npm install -g haas
$ houmous COMMAND
running command...
$ houmous (-v|--version|version)
haas/0.0.1 darwin-x64 node-v10.13.0
$ houmous --help [COMMAND]
USAGE
  $ houmous COMMAND
...
```
